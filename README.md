# Election Tally Board Application

![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB) ![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white) ![Status](https://img.shields.io/badge/Status-Discontinued-inactive?style=for-the-badge)

> **A web-based visualization tool designed to track election vote tallies.**

This application tracks vote counts to provide better visualization of election statistics, real-time candidate rankings, and status updates. It serves as a digital dashboard to make election data easier to digest and monitor.

**Project Status:**
This project is currently **Discontinued** and is no longer being actively maintained or updated. The code is archived here for portfolio and reference purposes.

**Backend Repository:**
[Tally Board Backend](https://github.com/rendznicoy/care-win-backend)

## Key Features

* **Data Visualization:** Graphical representation of voting statistics for quick analysis.
* **Live Leaderboards:** Real-time tracking of candidate rankings and vote distribution.
* **Responsive Dashboard:** Accessible interface for monitoring election progress on various devices.

## Tech Stack

* **Framework:** React
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **Package Manager:** npm
